# Flask Video Streaming App

This is a simple Flask app that streams video from a webcam. T
The app uses the OpenCV library to capture frames from the webcam and the Flask framework to serve these frames as a multipart/x-mixed-replace stream over HTTP.

## Requirements

Python 3.6 or higher
OpenCV 3.4 or higher
Flask 1.1 or higher
